Création d'un nouveau batch

En attendant d'avoir la tâche ant de création d'un nouveau projet (automatique), il convient de suivre les instructions suivantes pour créer un nouveau projet :
  1. Créer un nouveau projet Java
  2. Copier le contenu du dossier templates à la racine du nouveau projet
  3. Modifier le fichier build.properties
  4. Le build.xml (ant) doit alors être opérationnel
  5. La classe principale (exécutant le main) doit hériter de la classe GenericBatch
  6. Implémenter les méthodes abstraites

Cibles importantes du fichier ant

dist

Cette cible permet de construire le batch avec sa documentation.

Le dossier dist contiendra tous les éléments nécessaires à son exécution autonome.

run

Cette cible permet d'exécuter le batch.

new (TODO)

Cette cible permet de créer un nouveau projet basé sur GenericBatch.
